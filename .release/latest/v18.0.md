v18.0: Java 11 Compatibility & Unified 1.8.8 → 26.1.x Support
# GriefPrevention3D v18.0

**Wiki:** [GriefPrevention3D Wiki](https://github.com/castledking/GriefPrevention3D/wiki?utm_source=chatgpt.com)

## Highlights

* **Single build for all supported versions** — Minecraft 1.8.8 through 26.1.x now supported with one Java 11 build.
* **Major legacy compatibility fixes** — Improved claim tool behavior and far-away click handling on older Bukkit/Spigot servers.
* **Improved raycast accuracy** — Fixed water click offset issues and inconsistent targeting across versions.
* **Removed unstable packet injection path** — Disabled legacy Netty interception that could interfere with commands and interactions on 1.8.8.

## Compatibility

| Category           | Support        |
| ------------------ | -------------- |
| Minecraft Versions | 1.8.8 → 26.1.x |
| Minimum Java       | Java 11        |
| Bukkit API Target  | 1.13           |

## Fixes

### Legacy Servers (1.8.8–1.12.2)

* Fixed far-away golden shovel/stick interactions.
* Fixed duplicate claim creation paths.
* Fixed delayed/bursty camera update behavior.
* Fixed incorrect claim tool detection with unrelated items.
* Removed problematic Netty packet injection behavior.

### Modern Servers (1.13+)

* Fixed Material API compatibility issues.
* Fixed water click offset and fluid raycast inconsistencies.
* Refactored Java 21-only features for Java 11 compatibility.

## Migration

No data migration is required.

Existing claims, configs, and player data remain compatible.

**Recommended:** Test v18.0 in a production-clone or staging environment before deploying to a live server, especially on heavily customized or legacy setups.

## Pairing

Pairs with **GPExpansion v1.1.1** for claim block accrual profile controls.
