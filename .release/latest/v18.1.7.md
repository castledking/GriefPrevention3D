# GriefPrevention3D v18.1.7

**Wiki:** [GriefPrevention3D Wiki](https://github.com/castledking/GriefPrevention3D/wiki)

## Highlights

- **Cross-Claim Shaped Merge** — Use `/shapedclaims` to reshape your claim and merge it into a neighboring claim. Click the boundary of your claim, draw a path that overlaps the target claim, and close the path. The tool handles the merge automatically — overlapping land is claimed, disconnected components are bridged with a Manhattan path, and the target claim is deleted.
- **`/mergeclaims` Command** — Standalone command that toggles merge mode on your shovel. Visualize nearby claims with a sneak-click, then click the boundary of your target claim to select it. Works with `/claim mode merge` as well.
- **`/claim mode merge`** — New merge mode option for the shovel, accessible via `/claim mode merge`, `/mergeclaims`, or `/shapedclaims` (which enters merge mode automatically when drawing cross-claim paths).
- **Nearby Claim Visualization** — Nearby claims are now highlighted with visible borders during shaped edit and merge operations, making it easy to see which claims can be merged into.
